Een **Loop** (lus) is een controlestructuur die een stuk code **meerdere keren** uitvoert, totdat een bepaalde voorwaarde is voldaan.

Waarom zijn Loops essentieel?

Loops voorkomen dat je code steeds opnieuw moet typen (redundantie). Dit maakt je code:

  • **Effici√´nt:** Minder regels code.
  • **Schalerbaar:** Werkt gemakkelijk voor 10, 100 of 1000 items.

Voorbeeld: 25 studenten welkom heten

Zonder Loop vs. Met Loop:

// Zonder loop (25 regels nodig):
console.log("Welkom student 1");
console.log("Welkom student 2");
// ... enzovoort

// Met een loop (1 regel):
for (let i = 1; i <= 25; i++) {
    console.log("Welkom student " + i);
}

De Drie Soorten Loops

In JavaScript zijn er drie primaire loop-varianten, elk met een eigen toepassing:

Type Wanneer gebruiken? Hoofdkenmerk
For-loop Je **weet** hoeveel keer iets moet gebeuren (bijv. 10 keer, door een array). Initialisatie, Voorwaarde en Update staan op √©√©n regel.
While-loop Je **weet niet** hoeveel keer, maar checkt de voorwaarde vooraf. Blijft draaien zolang de voorwaarde true is.
Do-while-loop Je wilt dat de code **minimaal √©√©n keer** wordt uitgevoerd, dan pas checken. De check staat aan het einde van de lus.

De For-loop: De Standaard Werkpaard

De meest gebruikte lus, ideaal voor het itereren over een bekend aantal stappen.

Basis Syntax

for (/* 1. Initialisatie */; /* 2. Voorwaarde */; /* 3. Update */) {
    // Code die herhaaldelijk wordt uitgevoerd
}

Voorbeeld 1: Tellen van 1 tot 5

// 1. Start bij i=1; 2. Stop als i > 5; 3. i verhogen met 1 na elke keer.
for (let i = 1; i <= 5; i++) {
    console.log(i);
}
// Output: 1, 2, 3, 4, 5

Voorbeeld 2: Loop door een Array

const namen = ["Sarah", "Mohamed", "Lisa"];

for (let i = 0; i < namen.length; i++) {
    console.log("Hallo " + namen[i] + "!");
}
// Let op: Array-index begint bij 0, dus de voorwaarde is 'i < length'.

De While-Loop: Onbekend Aantal Iteraties

Gebruik deze lus als je **niet weet** hoe vaak de code moet worden uitgevoerd. De voorwaarde wordt **eerst gecheckt**.

Basis Syntax

while (/* Voorwaarde */) {
    // Code die herhaaldelijk wordt uitgevoerd
    // Vergeet hier niet de variabele te updaten!
}

Typisch Scenario: Gebruikersinvoer

De loop gaat door totdat de gebruiker een specifiek commando geeft (bijvoorbeeld "stop").

let invoer = '';

while (invoer !== 'stop') {
    invoer = prompt('Typ iets (of "stop"): ');
    console.log('Je typte: ' + invoer);
}

console.log('Programma gestopt!');

Onthoud het verschil met For

Bij een `while-loop` moet je de teller of de verandering van de variabele **zelf regelen** binnen de body van de lus om een oneindige loop te voorkomen.

De Do-While-Loop: Minimaal √â√©n Keer

Deze loop garandeert dat de code **minimaal √©√©n keer** wordt uitgevoerd, omdat de voorwaarde **pas achteraf** wordt gecheckt.

Basis Syntax

do {
    // Code wordt ALTIJD 1x uitgevoerd.
} while (/* Voorwaarde */);

Verschil met While-loop

Stel, de voorwaarde is direct onwaar:

let x = 10;

// 1. While-loop: Checkt eerst. De code wordt 0 keer uitgevoerd.
while (x < 5) {
    console.log("Nooit geprint (While)");
}

// 2. Do-while-loop: Voert uit, dan checkt. De code wordt 1 keer geprint.
do {
    console.log("Wordt 1x geprint (Do-While)");
} while (x < 5);

Ideaal voor situaties waarin je de gebruiker **eerst** iets wilt tonen of vragen voordat je controleert of er gestopt moet worden.
